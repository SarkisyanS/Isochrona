# ISOCHRONA APP
THE FULL DESCRIPTION WILL BE UPLOADED LATER

## Quick start
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Usage
1) Upload points (GeoJSON/GPKG).  
2) Provide roads (download from OSM or upload GeoJSON/GPKG).  
3) Build the graph.  
4) Set distances and boundary mode (buffer/concave).  
5) Run isochrones, view the map, and download GeoJSON.

## Notes
- Results are exported in WGS84 GeoJSON.
